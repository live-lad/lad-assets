# Licenciamento dos sons

Os sons ambientes (`type: "sound"` no `manifest.json`, servidos em `full/<id>.mp3`) foram
obtidos do projeto [Blanket](https://github.com/rafaelmardojai/blanket) (GPL-3.0) e
**transcodificados de `.ogg` para `.mp3`** para compatibilidade com os webviews do app
(WebKit no macOS/Linux não decodifica OGG Vorbis via Web Audio). Os arquivos de origem
estão em `source/<id>.ogg`.

Cada som mantém a licença original do autor. Sons `CC BY` e `CC BY-SA` exigem atribuição;
a cláusula share-alike dos `CC BY-SA` recai sobre o próprio arquivo de áudio, não sobre o
software que o reproduz.
